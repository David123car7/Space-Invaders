# Space-Invaders

Re-creation of Space Invaders game, built using  using C++ and the [raylib](https://github.com/raysan5/raylib) library.

![desc](docs/image.png)

# Build & Run

```bash
cd src
cmake -S . -B build
cmake --build build
./build/SpaceInvaders
```
